HTML (abreviação para a expressão inglesa HyperText markup Language, que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na contrução de páginas na Web. Documentos HTML podem ser intepretados por navegadores. wikipedia.org- acesso 09/2016

Tecnologia - Realidade Aumentada

De uma forma simples, Realidade Aumentada é uma tecnologia que prmite que o mundo virtual seja misturado ao real,, possibilitando maior interação e abrindo uma nova dimensão na maneira como nós executamos tarefas, ou mesmo as que nós incubimos às máquinas. Assim, se você pensava que objetos pulando para fora da tela eram elementos de fimes de ficção científica, está na hora de mudar seus conceitos. Aliás, o que acontece com a Realidade Aumentada é o contrário: você pulará para dentro do mundo virtual para interagir com objetos que só estão limitados à sua imaginação. tecmundo.com.br/realidade-aumentada - acesso 09/2016

Acessibilidade

Acessibilidade na web significa que pessoas com deficiência podem usar a web. Mais especificamente, a acessibilidade na web significa que pessoas com deficiência podem perceber, entender, navegar, interagir, contribuir para a web. Ela também benefícia outras pessoas, incluindo pessas idosas com capacidade em mudança devido ao envelhecimento - w3c.br/ - acesso em 09/2016
